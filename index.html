Welcome to ClusterIt

This Python web app can be used to provide spatially clustered features encoded in GeoJSON format, ready for consumption in OpenLayers or Leaflet for example.

Installation

If you want to use the app, the easiest way is to use a python virtualenv, I recommend using virtualenvwrapper as it makes working with virtualenvs just so much easier:

$ mkvirtualenv clusterit
$ mkdir myclusterit
$ pip install clusterit
$ cd myclusterit
$ wget http://geospatial.io/clusterit/files/settings.py

Go ahead and make changes to the settings module to reflect your database.

$ export CLUSTERIT_SETTINGS_MODULE=settings
$ clusterit.py

Now head over to your browser and open http://localhost:5000 to get a overview page.
